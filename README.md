# Tetris

An attempt to recreate Tetris.

## Building

> Make sure you are in the project's root directory.

```sh
cmake . # Configure
make # Compile
./tetris # Run
```

## Controls

You can control the game only using the keyboard.

### Menu

| Key                | Action   |
| ------------------ | -------- |
| `W`/`S` or `↑`/`↓` | Navigate |
| `Enter`            | Select   |
| `Q`                | Quit     |

## Settings

| Key                | Action                      |
| ------------------ | --------------------------- |
| `W`/`S` or `↑`/`↓` | Navigate                    |
| `A`/`D` or `←`/`→` | Adjust values               |
| `Enter`            | Select (only `Back` button) |
| `Q`                | Quit                        |

## Game

| Key     | Action            |
| ------- | ----------------- |
| `A`/`D` | Move horizontally |
| `S`     | Accelerate        |
| `J`/`L` | Rotate            |
| `Q`     | Quit              |
